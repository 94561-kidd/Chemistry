                                            • 金屬元素與化合物

                                            鹼金屬 ( IA族 )

                                            • 價電子組態均為 ns1，自然界中無元素態的鹼金屬存在
                                            • 除 Cs 為淺黃色， 其餘元素皆為銀白色
                                            • 需保存於石油、煤油或己烷中
                                            • 溶於水呈鹼性
                                            • 鹼金屬為最軟的金屬群，可用普通刀片切割
                                            • 熔點低 ( Li 最高: 186°C ； Fr 最低: 27°C)、比重小 ( Li、Na、K 皆小於 1 )
                                            • 化合物有焰色反應：Li (鮮紅)、Na (黃)、K (紫)、Rb (紫紅)、Cs (藍)

                                            製備

                                            電解熔融態的鹵化物：陰極得鹼金屬，陽極得鹵素分子

                                            MX(l)⟶M(s)+12XX2(g)

                                            當士法 (Downs process)：NaCl(l)⟶Na(s)+12ClX2(g)

                                            鹼金屬與非金屬

                                            鹵素

                                            • 形成離子化合物：

                                              Na+12ClX2⟶NaCl

                                            OX2

                                            • LiX2O
                                            • 低溫中形成 NaX2O，加熱會形成 NaX2OX2
                                            • KX2OX2、KOX2
                                            • RbOX2、CsOX2

                                            鹼金屬氧化物溶於水均呈鹼性

                                            NaX2O(s)+HX2O(l)⟶2NaOH(aq)

                                            NaX2OX2(s)+2HX2O(l)⟶2NaOH(aq)+HX2OX2(aq)

                                            HX2

                                            • 生成鹼金屬氫化物，放熱

                                              2Na+HX2⟶2NaH ( 200°C 以上才能發生反應 )

                                            • 鹼金屬氫化物只存在固態或熔融態，不能存在水溶液中

                                              NaH(s)+HX2O(l)⟶HX2(g)+NaOH(aq)

                                            HX2O

                                            • 生成鹼性氫氧化物和氫氣並放出巨大能量，原子序越大反應越激烈

                                              2Na(s)+HX2O(l)⟶2NaOH(aq)+HX2(g)

                                            酸或醇

                                            • 產生氫氣

                                              2Na+2CX2HX5OH⟶2CX2HX5ONa+HX2(g)

                                              2Na+2CHX3COOH⟶2CHX3COONa+HX2(g)

                                            鋰 ( Li )

                                            • Li 的原子半徑很小，因此性質與 Mg 相近

                                            • 鹼金屬化合物主要為離子鍵，但鋰化合物的鍵結有部分的共價性。

                                              LiCl、LiClOX4 可溶於乙醇、丙酮等有機溶劑。

                                            • 在水中最強的還原劑是 Li。( 水溶液中的還原性：Li > Rb > K > Cs > Na )

                                            • Li 的水合能為 121 kcal/mol，大於其他離子的水合能

                                            鈉 ( Na )

                                            NaCl ( 食鹽 )

                                            • 製備

                                              1. 粗鹽

                                                蒸發海水製得。

                                                含有 MgClX2 ( 苦味 )、CaClX2 ( 易潮解 )、MgSOX4、NaX2SOX4 等雜質

                                              2. 精鹽

                                                去除雜質後的高純度 NaCl。

                                                粗鹽溶於水，加入 NaX2COX3、BaClX2，過濾後加熱蒸發。

                                                MgClX2(aq)+NaX2COX3(aq)⟶MgCOX3(s)↓+2NaCl(aq)

                                                CaClX2(aq)+NaX2COX3(aq)⟶CaCOX3(s)↓+2NaCl(aq)

                                                MgSOX4+BaClX2(aq)⟶BaSOX4(s)↓+MgClX2(aq)

                                                NaX2SOX4+BaClX2(aq)⟶BaSOX4(s)↓+2NaCl(aq)

                                            • 性質

                                              1. 常溫下為無色透明立方晶體、有鹹味
                                              2. 易溶於水，溶於水呈中性
                                              3. 溫度對其溶解度影響甚小

                                            • 用途

                                              1. 調味、保存食物 ( 醃製 )
                                              2. 工業上常用以製造 Na、NaOH、NaX2COX3、ClX2、HCl、HClO

                                            NaOH ( 苛性鈉、燒鹼 )

                                            • 製備

                                              電解飽和濃食鹽水 ( 俗稱鹼氯工業 )。

                                              2NaCl(aq)+2HX2O→ 電解 2NaOH(aq)+HX2(g)+ClX2(g)

                                            • 性質

                                              1. 白色固體，極易溶於水

                                              2. 其水溶液有滑膩感，呈強鹼性

                                              3. 易吸收水分而潮解，亦可吸收 COX2

                                                2NaOH+COX2⟶NaX2COX3+HX2O

                                              4. 腐蝕性強，濃溶液能腐蝕玻璃，因此不能用玻璃盛裝濃 NaOH(aq)

                                              5. 與銨鹽共熱，可產生 NHX3

                                                NHX4Cl+NaOH→ △ NaCl+NHX3↑+HX2O

                                                (NHX4)X2SOX4+2NaOH→ △ NaX2SOX4+2NHX3↑+2HX2O

                                            • 用途

                                              1. 氫氧化鈉是實驗室中重要的強鹼試劑
                                              2. 工業上可製造肥皂、紙漿、人造絲、煉鋁工業、染料工業、食品工業等

                                            Na2CO3、NaHCO3

                                            • 別稱：

                                              1. NaX2COX3：蘇打、洗滌鹼、純鹼
                                              2. NaHCOX3：酸式碳酸鈉、小蘇打、焙用鹼

                                            • 製備 ( 索耳末法 Solvay's process ) ( 亦稱氨鹼法 )

                                              1. CaCOX3→ △ CaO+COX2
                                              2. CaO+HX2O⟶Ca(OH)X2
                                              3. COX2(g)+NHX3(g)+HX2O(l)⟶NHX4HCOX3(aq)
                                              4. NHX4HCOX3(aq)+NaCl(aq)⟶NHX4Cl(aq)+NaHCOX3(s)
                                              5. 2NHX4Cl+Ca(OH)X2⟶CaClX2+2NHX3+2HX2O
                                              6. 2NaHCOX3(s)→ △ NaX2COX3(s)+COX2(g)+HX2O(l)

                                              淨反應式 ( 原料僅為 CaCOX3、NaCl )

                                              CaCOX3+2NaCl⟶NaX2COX3+CaClX2

                                            • 性質

                                            • 用途

                                              1. NaX2COX3

                                                • 用於清洗 ( 例如：水壺 )
                                                • 軟化硬水
                                                • 製造玻璃、紙漿、清潔劑⋯⋯等

                                              2. NaHCOX3

                                                • 作為制酸劑，中和胃酸

                                                  NaHCOX3+HCl⟶NaCl+COX2+HX2O

                                                • 和酒石酸氫鉀混合可成為做麵包和餅乾時所用的焙粉 ( 發粉 )，受熱產生 　COX2 使麵粉膨鬆

                                                • 和硫酸混合可以當滅火劑

                                                  2NaHCOX3+HX2SOX4⟶NaX2SOX4+2COX2(g)+2HX2O

                                            鉀 ( K )

                                            KNO3 ( 硝石 )

                                            • 製備

                                              1. 自然界的礦物

                                              2. 人工合成法：以 NaNOX3 ( 智利硝石 ) 和 KCl(aq) 共熱

                                                2NaNOX3(aq)+KCl(aq)⟶KNOX3(aq)+NaCl(aq)

                                                先煮沸析出 NaCl，再冷卻析出 KNOX3 結晶。

                                            • 性質

                                              1. 無色晶體，易溶於水

                                              2. 為良好的氧化劑，加熱至 334°C 時，會分解成 OX2、NX2 和 KX2O

                                                2KNOX3(s)→ △ 2KNOX2(s)+OX2(g)

                                                4KNOX2(s)→ △ 2NX2(g)+3OX2(g)+2KX2O(s)

                                            • 用途

                                              1. 製造火藥及煙火

                                              2. 作為肥料，可同時提供鉀肥、氮肥

                                            K2CO3 ( 草鹼 )

                                            • 製備

                                              將 COX2 通入 KOH

                                            • 用途

                                              1. 製造玻璃、肥皂、KOH 的原料
                                              2. 草木灰約含 10% KX2COX3，因此草木灰可供洗滌之用
                                              3. 製造肥料，K 是植物養料三要素之一，因此古代常用草木灰當作肥料

                                            KOH ( 苛性鉀 )

                                            • 製備

                                              電解濃 KCl(aq)

                                            • 性質

                                              1. 白色固體、易潮解
                                              2. 極易溶於水，為強鹼，有滑膩感
                                              3. KOH(aq) 易吸收空氣中的 COX2，形成 KX2COX3

                                            • 用途

                                              和硬酯酸反應產生硬酯酸鉀，為一種軟肥皂

                                            鹼土金屬 ( IIA族 )

                                            • 價電子組態均為 ns2，自然界中無元素態的鹼金屬存在
                                            • 稱為鹼土族是因為高熱不熔、不易分解，有「土」的性質
                                            • Be 活性小，高溫亦不和水反應；Mg 會和沸水、水蒸氣反應
                                            • Ca、Sr、Ba 需保存於石油中
                                            • 金屬晶體結構不同，因此金屬鍵長度、熔點、沸點、密度、昇華熱、鍵能、導電性等物理性質皆無規律性變化
                                            • 金屬鍵比同週期的鹼金屬強，因此熔點、硬度皆比同週期的鹼金屬大
                                            • 隨著原子序增大，IIA族元素的還原力漸增
                                            • 同一級游離能隨原子序增加而遞減
                                            • 第一游離能 IE1：第二游離能 IE2 = 1：2
                                            • 氧化電位 E0 隨原子序增加而遞增
                                            • 化合物有焰色反應：Ca (磚紅)、Sr (深紅)、Ba (黃綠)

                                            鹼土金屬與非金屬

                                            HX2O

                                            • 氧化物溶於水呈鹼性，為放熱反應，ΔH 隨原子序增加而漸增

                                              CaO(s)+HX2O⟶Ca(OH)X2      ΔH=−15.6kcal/mol

                                            • 氫化物溶於水產生氫氣

                                              CaHX2(s)+2HX2O(l)⟶Ca(OH)X2(aq)+2HX2(g)

                                            • 與水反應產生氫氣

                                              CaHX2(s)+2HX2O(l)⟶Ca(OH)X2(aq)+2HX2(g)

                                            OHX−

                                            • 鹼性：Ba(OH)X2 > Sr(OH)X2 > Ca(OH)X2 > Mg(OH)X2 > Be(OH)X2

                                            • 溶解度：Ba(OH)X2 > Sr(OH)X2 > Ca(OH)X2 > Mg(OH)X2 > Be(OH)X2

                                            • Be 為兩性元素，Be(OH)X2 可溶於酸和鹼

                                              Be(OH)X2+2HX2X+⟶BeX2++2HX2O

                                              Be(OH)X2+2OHX−⟶Be(OH)X4X2−

                                            COX3X2−

                                            • 碳酸鹽受熱分解產生 COX2

                                              CaCOX3(s)→ △ CaO(s)+COX2(g)
